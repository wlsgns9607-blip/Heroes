정서적 보상 없음

생명을 구하고도 감사받지 못하는 환경. 정서적 번아웃이 직업 만족도를 무너뜨리고 있습니다.

🚫

사회적 리스펙 부족

현장 출동 시 사회적 존중 부재. 시민과의 갈등이 직무 스트레스로 이어집니다.

📁

출동 현황 관리 공백

기존 시스템은 출동 데이터와 소방관 개인 정보를 연동하지 못해 업무 효율이 낮습니다.

MVP 핵심 기능

네 가지로
완성됩니다

01

소방관 전용 출동 데이터 입력 Form

현장에서 즉시 입력 가능한 10대 출동 유형 드롭다운 — 건축물화재, 구조·구급, 교통사고, 승강기 갇힘 등. 빠르게 실적을 기록하는 소방관 전용 인터페이스.

📋
02

출동 & 진압 성공 시 명예 배지 수여

화재 진압 마스터, 인명 구조 성공 시 '❤️ 생명의 은인', 시민 피드백 기반 '🏅 국민이 인정한 영웅' 배지 포함. 정서적 보상을 시각화하는 게이미피케이션 시스템.

🏅
03

시민 감사 & 리스펙 댓글 UI

일반 시민이 소방관에게 익명으로 감사 댓글을 남기고, 커피 쿠폰을 발송할 수 있는 소방관 현장 댓글 UI. 진심이 연결되는 소통 창구.

💬
04

🎙️ 음성 인식 출동 기록 입력

현장에서 손을 쓸 수 없는 소방관이 목소리만으로 출동 기록을 완성하는 핸즈프리 입력 시스템입니다.

마이크 버튼 하나를 누른 뒤 "건축물 화재, 인명 구조 2명, 진압 완료"라고 말하면 Web Speech API가 실시간으로 음성을 텍스트로 변환해 출동 유형·결과·특이사항 필드를 자동으로 채웁니다.

소방 현장 용어(진압, 구조, 구급, 잔화정리 등)에 최적화된 커스텀 사전을 적용해 인식 정확도를 높이고, 인식된 텍스트는 즉시 화면에 표시되어 대원이 눈으로 확인 후 한 번의 탭으로 저장할 수 있습니다.

장갑을 낀 상태, 연기·소음이 심한 환경을 고려해 노이즈 필터링을 적용하며, 음성 입력이 어려운 경우 기존 드롭다운 Form으로 즉시 전환할 수 있는 폴백(Fallback) 모드를 제공합니다.

🛠 사용 기술: Web Speech API (SpeechRecognition) · 소방 도메인 커스텀 Grammar · 실시간 transcript 미리보기 · 노이즈 필터링 후처리
